# PyTorch YOLOv4 DeepSort 项目

基于 PyTorch 实现的 YOLOv4 目标检测与 DeepSORT 多目标跟踪项目。该项目将 YOLOv4 目标检测器与 DeepSORT 跟踪算法相结合，能够在视频或实时流中实现目标的检测与 ID 持续跟踪。

## 项目简介

PyTorch_YOLOv4_Deepsort 是一个将 YOLOv4 目标检测与 DeepSORT 多目标跟踪进行深度整合的开源项目。该项目采用 PyTorch 作为深度学习框架，能够对视频画面中的目标进行实时检测，并为每个检测到的目标分配唯一的跟踪 ID，从而实现目标轨迹的连续追踪。

该项目的设计目标是提供一个简单易用、性能优异的实时目标检测与跟踪解决方案。通过结合 YOLOv4 的强大检测能力与 DeepSORT 的先进跟踪算法，用户可以快速构建自己的目标跟踪应用，无需从头实现复杂的算法逻辑。

## 功能特性

本项目具有以下核心功能：

### 1. YOLOv4 目标检测

项目集成了 YOLOv4 目标检测器，这是目前最先进的目标检测算法之一。YOLOv4 在检测精度和速度之间取得了出色的平衡，能够识别多种类别的目标，包括人物、车辆、动物等常见物体。检测器使用了 CSPDarknet53 作为主干网络，并采用了多种先进的技术来提升检测性能，包括 PANet、SPP 等模块。

### 2. DeepSORT 多目标跟踪

DeepSORT 是基于深度学习的多目标跟踪算法，它在传统 SORT 算法的基础上引入了外观特征描述子，能够有效处理目标遮挡和短暂消失的情况。每个被跟踪的目标都会维护一个特征向量，当目标重新出现时，系统会通过特征匹配来判断是否是之前跟踪的同一目标，从而实现更加稳定和准确的跟踪效果。

### 3. 目标 ID 持续追踪

系统会为每个检测到的目标分配唯一的跟踪 ID，并在目标移动过程中保持该 ID 不变。即使目标暂时被遮挡或离开画面，再次出现时仍有机会恢复之前的跟踪 ID。这对于分析目标行为、统计流量等应用场景非常重要。

### 4. 视频流处理支持

项目支持处理本地视频文件，用户可以将待分析的视频放入指定的目录中，系统会自动进行目标检测与跟踪。同时，代码结构也支持扩展到实时视频流的应用场景，如摄像头输入或网络视频流。

### 5. 实时检测与跟踪

经过优化的代码实现能够在配备适当硬件的设备上实现实时或接近实时的处理速度。系统支持 GPU 加速，通过 CUDA 可以大幅提升推理速度，满足实际应用的需求。

## 技术架构

### 检测模型

项目采用 YOLOv4 作为目标检测模型，这是 YOLO 系列算法的第四代版本。YOLOv4 在 YOLOv3 的基础上进行了大量的改进，包括使用 CSPDarknet53 作为主干网络、引入 PANet 进行特征融合、采用 SPP 模块增加感受野等技术。这些改进使得 YOLOv4 在 COCO 数据集上达到了极高的检测精度，同时保持了较快的推理速度。

### 跟踪算法

DeepSORT 是本项目采用的跟踪算法，其核心思想是将目标检测与卡尔曼滤波相结合，并引入深度学习提取的外观特征进行目标关联。DeepSORT 使用一个轻量级的卷积神经网络来提取目标的表观特征，在跟踪过程中维护一个特征库，通过余弦相似度来判断目标身份。

### 深度学习框架

项目基于 PyTorch 深度学习框架实现，PyTorch 提供了灵活的动态图机制和丰富的工具库，使得模型的构建、训练和部署都变得非常简单。项目同时提供了 CUDA 10.1 的配置文件，方便用户在支持 CUDA 的环境中启用 GPU 加速。

## 安装步骤

### 环境要求

在开始安装之前，请确保您的系统满足以下要求：

- Python 3.7 或更高版本
- PyTorch 1.6.0 或更高版本
- CUDA 10.1 或更高版本（可选，用于 GPU 加速）
- 8GB 以上内存（推荐）
- 支持 CUDA 的显卡（可选，用于 GPU 加速）

### 安装依赖

首先，克隆项目到本地：

```bash
git clone https://github.com/VeridianByte/PyTorch_YOLOv4_Deepsort.git
cd PyTorch_YOLOv4_Deepsort
```

然后，使用 pip 安装项目依赖：

```bash
pip install -r requirements.txt
```

如果您的环境使用 CUDA 10.1，可以使用提供的 conda 环境配置文件创建环境：

```bash
conda env create -f cuda101.yml
conda activate cuda101
```

### 下载预训练权重

项目需要 YOLOv4 的预训练权重文件才能运行。请从官方渠道下载预训练权重，并将其放置在 `weights/` 目录中。权重文件的详细获取方式请参考项目文档或相关资源。

## 使用方法

### 基本使用

项目提供了简洁的运行接口，您可以通过以下命令启动目标检测与跟踪：

```bash
python run.py
```

### 处理视频文件

将待处理的视频文件放入 `videos/` 目录中，程序会自动识别并处理该目录中的视频文件。处理完成后，结果会保存到指定位置，您可以在输出中查看检测和跟踪的结果。

### 参数配置

项目提供了多个可配置的参数，您可以根据实际需求调整检测阈值、跟踪参数等设置。具体参数说明请参考代码中的配置部分或官方文档。
